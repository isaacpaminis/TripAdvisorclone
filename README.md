TripAdvisor Frontend Clone

Overview

A responsive TripAdvisor-inspired website developed using HTML and CSS.
This project replicates the visual structure and layout of the original platform while focusing on clean UI implementation and responsive design principles.

Technologies Used

HTML5
CSS3
Flexbox
Media Queries

Responsive Design

The layout adapts seamlessly across different screen sizes, ensuring usability and visual consistency on both desktop and mobile devices.

Outcome:

Strong understanding of layout systems
Responsive web design implementation
Maintainable and structured CSS styling
